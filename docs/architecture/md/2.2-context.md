## 2.2 System Context
Below is the context diagram for the ClassiCam system.

<div style="display: flex; justify-content: center;"><div style="width:700px">

```mermaid
graph TB

    subgraph Input Sources
        camera[Camera]
        photo_lib[Photo Library]
    end
    camera --> |captures new images| ClassiCam
    photo_lib --> |provides stored images| ClassiCam

    subgraph AI Processing
        gpu[GPU]
        ai_model[AI Models]
        model_mgr[Model Manager]
    end
    gpu --> |accelerates model inference| ClassiCam
    ai_model --> |loads / runs models| ClassiCam
    model_mgr --> |manages model lifecycle| ClassiCam

    ClassiCam((ClassiCam Application))

    results[Classification Results]
    ClassiCam --> |generates and exports| results

    subgraph Results History
        results_db[Results Database]
        hist_viewer[History Viewer]
    end
    ClassiCam --> |stores history| results_db
    ClassiCam --> |displays past results| hist_viewer

    subgraph Extneral Integrations
        cloud[Cloud Storage]
        mobile_app[Mobile Apps]
        web_app[Web Apps]
    end
    ClassiCam --> |syncs data remotely| cloud
    ClassiCam --> |sends data / launches| mobile_app
    ClassiCam --> |sends / receives data| web_app

```
