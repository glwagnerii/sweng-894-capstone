Here is a high level state machine for the test automation framework / use cases...

<div style="display: flex; justify-content: center;"><div style="width:600px">

```mermaid
stateDiagram-v2
    [*] --> Idle
    Idle --> ExecutingTestCase: startTestExecution()
    ExecutingTestCase --> CollectingData: collectData()
    CollectingData --> AnalyzingResults: analyzeResults()
    AnalyzingResults --> GeneratingReport: generateReport()
    GeneratingReport --> [*]: reportComplete
    
    ExecutingTestCase --> Paused: pauseExecution()
    Paused --> ExecutingTestCase: resumeExecution()
    Paused --> Idle: stopExecution()
    
    ExecutingTestCase --> Idle: stopExecution()
    CollectingData --> Idle: stopExecution()
    AnalyzingResults --> Idle: stopExecution()
    GeneratingReport --> Idle: stopExecution()
```
